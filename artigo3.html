1. O que é Midjourney?

3. Como Usar Midjourney

Comando Básico

Para gerar uma imagem, use o comando:

/imagine prompt: sua descrição aqui

Exemplo Prático

/imagine prompt: um homem trabalhando em um computador em um escritório moderno, iluminação natural, estilo fotográfico, 4k

Midjourney gerará 4 variações. Você pode:

  • U1-U4: Fazer upscale (aumentar qualidade) de uma imagem
  • V1-V4: Criar variações de uma imagem
  • Regenerate: Gerar 4 novas imagens

4. Estrutura de um Bom Prompt

A qualidade da imagem depende de como você escreve o prompt. Aqui está a estrutura ideal:

/imagine prompt: [OBJETO] [DESCRIÇÃO DETALHADA] [ESTILO] [ILUMINAÇÃO] [QUALIDADE] [PARÂMETROS]

Exemplo Completo

/imagine prompt: uma mulher em um café, bebendo café, sorrindo, cabelos castanhos, estilo fotográfico realista, iluminação natural dourada, 4k, alta qualidade, profissional --ar 16:9 --q 2

Componentes do Prompt

  • Objeto: O que você quer (mulher, paisagem, produto, etc.)
  • Descrição: Detalhes (cor, tamanho, posição, ação)
  • Estilo: Tipo de arte (fotográfico, ilustração, pintura, etc.)
  • Iluminação: Como deve ser iluminado (natural, dramático, suave)
  • Qualidade: Resolução (4k, 8k, ultra HD)
  • Parâmetros: Configurações especiais (--ar, --q, --s)

5. Parâmetros Importantes

Parâmetro O que faz Exemplo
--ar Define a proporção (aspect ratio) --ar 16:9 (paisagem)
--q Define a qualidade (0.25 a 2) --q 2 (máxima qualidade)
--s Define a "criatividade" (0 a 1000) --s 750 (mais criativo)
--niji Estilo anime/manga --niji (para anime)
--v Versão do modelo --v 6 (versão mais recente)

7. Dicas para Melhores Resultados

✅ Faça:

  • Seja específico e detalhado
  • Use palavras-chave de estilo (fotográfico, ilustração, pintura)
  • Mencione a iluminação desejada
  • Especifique a qualidade (4k, 8k)
  • Use parâmetros para controlar o resultado
  • Teste diferentes variações

❌ Evite:

  • Prompts muito vagos ("uma imagem bonita")
  • Pedir coisas impossíveis (violar direitos autorais)
  • Usar muitas palavras-chave conflitantes
  • Esquecer de especificar o estilo
